Day 1 CodeWorks

  • GIT:
    A source control tool for versioning and storing projects or repositories.
  • Node: A javascript environment for working with projects.
  • npm: A package manager for the Node environment that allows you to find, install, and utilize different tools.
  • http-server: A server emulation tool within Node.
  • VSCode: Software that allows you to edit and write code within a project.
  • Setup Source Directory: A set area to store projects locally for easy indexing and workflow.
